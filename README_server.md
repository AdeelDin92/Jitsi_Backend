# Getting Started with Node Server 

In the project directory, you can run:

### `node server.js`

This will spin up the server but currently server is deployed on cloud "Cyclic" . 

For local testing please change the post URL from client to "http://localhost:3000/" . Note the localhost port can be changed to any open port . Here it is 3000 .

Please check the axios.post for POST URL from client
